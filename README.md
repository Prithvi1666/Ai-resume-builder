## 📌 Overview

AI Resume Builder is a modern web application designed to help users generate professional resumes effortlessly using Artificial Intelligence. The platform provides smart content suggestions, elegant resume templates, live previews, and easy PDF downloads.

Whether you are a fresher, student, or experienced professional, this platform helps you build an impressive resume quickly and efficiently.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Prithvi1666/Ai-resume-builder.git
```

## 2️⃣ Navigate to Project Directory

```bash
cd Ai-resume-builder
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Configure Environment Variables

Create a `.env` file in the root directory and add:

```env
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
AI_API_KEY=your_ai_api_key
PORT=5000
```

## 5️⃣ Start Development Server

```bash
npm run dev
```
